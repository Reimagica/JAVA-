# 要求
老师给定科技文档MySql数据表，基于该数据表建立术语之间的关系网络。在关系网络的基础上，基于Lucene或ElasticSearch搭建一个B/S模式信息检索系统，实现全文检索和术语关系展示。
【数据表由老师后期提供】
